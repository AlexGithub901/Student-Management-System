### Most Used Commands in This Project
```bash
git checkout -b feature-export          # Create feature branch
git add .                               # Stage all changes
git commit -m "message"                 # Commit with message
git push origin main                    # Push to remote
git pull origin main                    # Pull latest
git status                              # Check status
git log --oneline                       # View history
git merge feature-export                # Merge branch
```

### Complete Feature Branch Workflow
```bash
git checkout -b feature-export          # 1. Create branch
# ... make changes ...
git add .                               # 2. Stage
git commit -m "description"             # 3. Commit
git push origin feature-export          # 4. Push
# Create PR on GitHub
git pull origin main                    # 5. Pull latest
git add .                               # 6. Resolve if needed
git commit -m "Merge conflicts"         # 7. Commit
git push origin main                    # 8. Push
```

### Conflict Resolution
```bash
git status                              # See conflicts
# Edit files manually
git add <file>                          # Stage resolved
git commit -m "Resolve conflicts"       # Commit
```
